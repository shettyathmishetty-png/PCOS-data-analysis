
# PCOS Data Analysis
This project is an Exploratory Data Analysis (EDA) on a PCOS dataset using Python. The goal is to understand patterns and relationships between different health factors related to PCOS.

## Dataset
- Age
- BMI
- Testosterone level
- Menstrual irregularity
- Antral follicle count
- PCOS diagnosis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Observations
- Higher BMI is observed in PCOS cases
- Testosterone levels are higher in individuals with PCOS
- Menstrual irregularity is strongly linked with PCOS
- Antral follicle count is higher in PCOS patients

## Conclusion
The analysis shows strong relationships between hormonal imbalance and PCOS diagnosis.
